# Remote Controller

簡易的遙控器，適合在床上觀看影片時使用


架構
---
* `Java JNA:` 呼叫Windows API (user32.dll) 發送按鍵事件
* `Java Servlet:` 後端接收按鍵觸發
* `Bootstrap:` 簡易的網頁按鍵介面
